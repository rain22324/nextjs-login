## 💡 常见问题

### Q: 点击登录后没有反应？
A: 
1. 检查浏览器是否阻止了弹出窗口
2. 检查控制台是否有错误
3. 访问 `/auth/debug` 查看详细状态

### Q: 登录成功但看不到仪表板？
A: 
1. 确保浏览器 Cookie 已启用
2. 尝试清除浏览器缓存
3. 检查 `/auth/debug` 页面的会话信息

### Q: 显示"认证出错"？
A:
1. 记下具体的错误信息
2. 检查 GitHub 回调 URL 是否正确
3. 确认 GitHub 提供商在 Supabase 中已启用
